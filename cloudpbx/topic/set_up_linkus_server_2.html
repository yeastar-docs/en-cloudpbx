Set up Linkus Server

Before users can log in Linkus clients, you should enable the Linkus server.

Enable Linkus Server

  1. Log in the PBX web interface, go to Linkus, check the option Enable.

  2. On the pop-up dialog, click Yes to confirm.
  • Linkus Login Information
    Users can log in Linkus by themselves after they receive the Linkus emails. You can also provide the detailed Linkus login information for them to enter on the Linkus Login Page.
