Set up Linkus Server with LCS

LCS (Linkus Cloud Service) avoids the necessity of port forwarding when using Linkus outside of the company so the network security will not be compromised. With LCS (Linkus Cloud Service), you can easily and quickly set up your Linkus server and enable Linkus Chat function for the users.

  1. Go to Linkus > Linkus Server Settings.
  2. To enable Linkus server, check the option Enable Linkus Server.
  3. To set up Linkus server with LCS, check the option Enable Linkus Cloud Service.
  4. On the pop-up window, click Yes to confirm.
  5. To enable Linkus Chat function for users, check the option Enable Chat function for Linkus.
