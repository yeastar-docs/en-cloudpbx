Linkus Server

Release notes for Linkus Server on Yeastar Cloud PBX Instance.

Version 2.1.7

Released Date: December 10, 2018

  • Fixed known issues.

Version 2.0.15

Released Date: October 26, 2018

  • Improved the Linkus stability.
  • Added support for forcing logout of Linkus desktop client on Linkus mobile client.

Version 1.8.15

Released Date: August 13, 2018

  • Users can log in Linkus mobile client and Linkus desktop client by a same login link.

Version 1.8.10

Released Date: July 26, 2018

  • Added support for Linkus Windows desktop.
  • Added support for Chat feature on Linkus.
  • Added support for Presence feature on Linkus.
  • Improved the stability of Linkus mobile client.
  • Optimized Linkus iOS push.
  • Improved the calling experience on Linkus Android client.
  • Fixed known issues.

Version 1.7.18

Released Date: June 29, 2018

  • Fixed the issue that Linkus call push would fail.

Version 1.7.17

Released Date: May 23, 2018

  • Optimized the Linkus login link.
  • Optimized the Linkus email template.

Version 1.6.10

Released Date: April 10, 2018

  • Users could only log in Linkus client by email address.
  • Improved the security of the Linkus login.
  • Expand the expiration time of QR code and login link that are in the Linkus email to 24 hours.
  • Optimized the Linkus Email Template.

Version 1.4.18

Released Date: January 10, 2018

  • Compatible with Yeastar Cloud PBX v81.3.0.12.

Version 1.4.12

Released Date: December 5, 2017

  • Linkus Server is enabled by default.
  • Added support for selecting all extensions and sending Linkus e-mails to all the extensions.
  • Optimized call push feature of Linkus client.
  • Optimized CDR on Linkus client: the CDR will display numbers without outbound prefixes.
  • Fixed the issue that there was noise when using Linkus client to play system prompts.
  • Fixed the issue that if Linkus call push was failed, the CPU usage of Linkus server was high.
  • Fixed the issue that if the web debug log was enabled on the PBX, Conference function on Linkus client could not work.
  • Fixed the issue that after re-enabling Linkus server, the ring strategy on Linkus client could not work.
  • Fixed the issue that after re-enabling Linkus server, users could not make outbound calls through Linkus client.
  • Fixed the Linkus login e-mail issue: if the email subject contains Chinese characters, the email contents would display messy codes.
