Auto Provision Phone Firmware

You can auto provision firmware of your phones.
Note: Auto provisioning phone firmware only works for the phones that were registered to the PBX via auto provisioning.
  1. Go to Auto Provisioning > Firmware Upgrade, the page displays the auto provisioned phone models.
  2. Click Check for New Version.
  3. Click the Upgrade To drop-down menu, check if the phones have new firmware version available.
    If the phone has new firmware version, you can see the version in the drop-down menu.
  4. Select a firmware version that you want to upgrade to.
  5. Reboot the phone(s) and update the firmware.
