Import Parameters - Blacklist/Whitelist

Descriptions for the Blacklist parameters and Whitelist parameters that you can import to Yeastar Cloud PBX.

Blacklist

Parameter Description Importance Restriction Default Value
Name Blacklist Name Required The following characters are NOT allowed:

Spaces and : ! $ ( ) / # ; , [ ] " = < > & ' ` ^ % @ { } |

 N/A
Number Numbers in Blacklist. Required Only the following characters are allowed:

Numbers and X x Z z N n W w . ! [ ] ^

For multiple numbers, enter numbers and use ^ as a separator, e.g. 13489999999^13488888888^13787777777.

Note: Enter at least one number. Or import will be failed.
 N/A
Type Blacklist Type Required Permitted value:

inbound outbound both

 inbound
Outlist Extensions Apply To
  • If Type is outbound, Outlist is required filed.
  • If Type is inbound or both, Outlist is optional field.
  • Enter Allto apply blacklist to all extensions.
  • Enter [EXTENSION_NUMBER] to apply blacklist to specific extensions.

    For multiple extensions, enter numbers and use & as a separator, e.g. 1000&1001&1002.

 N/A
